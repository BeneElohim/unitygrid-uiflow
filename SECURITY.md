# Security Policy

## Supported Versions

As unitygrid doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain: 
* main branch for unitygrid 2.x
* main-1.x branch for unitygrid 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/unitygrid/security to report the vulnerability you find.
